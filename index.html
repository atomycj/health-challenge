🔧 Firebase 설정 필요
index.html 파일 상단의 firebaseConfig를 실제 값으로 교체해주세요.
설정 방법
1. Firebase Console → 프로젝트 선택
2. ⚙️ 프로젝트 설정 → 앱 추가(웹) → firebaseConfig 복사
3. index.html 상단 firebaseConfig 객체에 붙여넣기
4. GitHub에 push → Vercel이 자동 재배포
Realtime Database 규칙
Firebase Console → Realtime Database → 규칙 탭에 아래를 붙여넣고 게시:

{ "rules": { ".read": true, ".write": true } }
